ABOUT

A multidisciplinary journey—from architectural design to systems programming—highlighting a passion for both spatial logic and computational thinking. It includes software projects, reflecting deep engagement with low-level programming and systems architecture. In parallel, selected architectural works explore the expressive potential of space, structure, and technology. Together, these works represent a continuous exploration of structure, function, and creativity—across both the digital and physical realms.



shuwang

Former architect turned software developer, with a background in spatial design and a growing expertise in system-level programming. After years of crafting physical spaces, I transitioned into tech to build digital systems that are equally structured and expressive. Currently studying at Ecole 42 in Paris, I enjoy working on network programming, shell environments, and real-time rendering engines. I’m particularly interested in projects that merge logic and creativity—whether in code or in space.

PROJECT

Aether PONG(2025)

Fullstack Pong-style online multiplayer web game, created by a team of 5. built with a microservice architecture, WebSocket real-time communication, and secure authentication, featuring 3D rendering with Babylon.js.

webserv(2025)

A lightweight HTTP server built from scratch by a team of 2. This project provided hands-on experience in socket programming, I/O multiplexing, HTTP request parsing, response generation, and back-end CGI script execution.

CNN_digit_recognition

The project implements a simple Convolutional Neural Network (CNN) in PyTorch for handwritten digit recognition, with key components such as convolution, pooling, activation functions, and fully connected layers.

MLP_breast_cancer_cell_classifier

Multilayer perceptron is a neural network implemented from scratch for non-linear regression and classification, applied in tasks such as breast cancer cell classification, supporting SGD and mini-batch training, real-time visualization, and so on.

DSLR_multi_class_classifier

DSLR implements a modular logistic regression classifier in Python to predict Hogwarts houses using a one-vs-all approach, with data analysis and visualization to explore the dataset and model behavior.

linear_regression(2025)

Linear Regression model implemented from scratch using gradient descent to predict car prices based on mileage, with Rust for core algorithm and Python for visualization.

cub3d(2024)

A 3D raycasting game engine that allows players to navigate through a 3D environment. It implements algorithms to render 3D scenes in a 2D space, similar to how older first-person games like Wolfenstein 3D worked.

minishell(2024)

A shell implementation by a team of 2. It involves parsing user commands, handling built-in functions, and managing process execution, signals, and redirections, closely mimicking a real Unix shell.

philosopher(2024)

The Philosopher project simulates the classic Dining Philosophers problem using threads in concurrent environment. Each philosopher alternates between thinking and eating, trying to pick up shared forks without causing deadlocks and data races.

so_long(2024)

A graphical 2D game project, it uses the MiniLibX library and focuses on implementing a 2D game environment where players can navigate through a maze.

push_swap(2024)

An algorithmic project from the Ecole42 curriculum. The goal is to sort a stack of numbers using a limited set of operations while optimizing for the least number of moves.

PISCINE

A comprehensive Python learning module focused on data science fundamentals. Covers Python programming, data analysis, visualization, and basic statistics concepts, with hands-on exercises using NumPy, Pandas, and Matplotlib. Ideal for building a solid foundation in Python for real-world data tasks.

Piscine C++(2024)

A C++ bootcamp focused on object-oriented programming and efficient code design. Implements features such as memory management, modular structures, and practical problem-solving techniques, highlighting core C++ skills and application in real programs.

Piscine C(2024)

A selection bootcamp in C language for hands-on practice and algorithm implementation, my starting point of programming journey. Covers fundamental programming concepts, manual memory management, pointers, and data structures, emphasizing building a strong foundation in C and problem-solving skills.
